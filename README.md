# LOADING | Team Portfolio

A modern, responsive portfolio website for the LOADING digital agency. This project showcases a clean design system, interactive elements, and a professional layout.

## Features

- **Responsive Design**: Fully responsive navigation and layout, including a custom mobile hamburger menu.
- **Interactive Hero Section**: Features a mouse-tracking parallax effect with depth for a dynamic first impression.
- **Custom Design System**: Consistent use of CSS variables for brand colors (Red, Blue, Yellow), typography (Space Grotesk), and spacing.
- **Bento Grid Layout**: A modern grid layout for the "Selected Work" section.
- **Page Transitions**: Custom page loader animation.
- **Animations**: CSS animations for entrance effects, floating shapes, and hover states.

## Project Structure

- `index.html`: The main landing page containing the Hero, Selected Work, Services, and Contact sections.
- `about.html`: The "About Us" page featuring the company story, team members, and values.
- `index.css`: Core styles, variables, reset, and styles for the main landing page.
- `about.css`: Specific styles for the About page.
- `img/`: Directory for images and assets (e.g., `loading-logo.svg`).

## Setup & Usage

1. Clone or download the repository.
2. Open `index.html` in any modern web browser.
3. No build step or installation required (Vanilla HTML/CSS/JS).

## Technologies

- **HTML5**: Semantic markup.
- **CSS3**: Flexbox, Grid, Custom Properties (Variables), Animation.
- **JavaScript**: ES6+ for DOM manipulation and interactive effects.
